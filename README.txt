Вот что я могу:
# Раздел начальных команд
/start - В НАЧАЛО - команда которая запускает меня в работу.

# Раздел погодных команд
/weather - ПОГОДА - команда котороя сообщит тебе погоду.
(Можно указать город, например "Симферополь" либо со смартфона отправить данные геолокации)

# Раздел команд для списков 
/my_list - СПИСКИ - команда которая открывают меню работы со списками.
/all_plist - ВСЕ СПИСКИ - команда которая показывает все добавленные списки.
/plist - ДОБАВИТЬ СПИСОК - команда которая добовляет новый список.
/see_last_plist - ПОСЛЕДНИЙ СПИСОК - команда которая показывает последний список.

# Раздел команд для шоу на YouTube и VKvideo
/my_show - ШОУ - команда которая открывают меню работы с шоу.
/all_show - ВСЕ ШОУ - команда которая показывает все добавленные шоу.
/add_new_show - ДОБАВИТЬ ШОУ - команда которая добовляет новое шоу.
/new_show - НОВЫЕ ШОУ - команда которая показывает новые шоу.

# Раздел помощь
/help - СПРАВКА - команда котороя расскажет обо мне.
/cancel - команда которая возвращает в начало.
